# Using the CLI Installer

## The Installation Process

**Basic example:**

```shell
npx makeswift@latest init
```

### Choose a Package Manager

```shell
yarn dlx makeswift@latest init
```

```shell
pnpx makeswift@latest init
```

```shell
bunx makeswift@latest init
```

### Variant Code Examples

**Template option:**

```shell
npx makeswift@latest init --template basic-typescript-pages
```

**Example option:**

```shell
npx makeswift@latest init --example basic-typescript-pages
```

## Edge Scenarios

### Can I Just Get the Code?

**Using degit:**

```shell
npx degit https://github.com/makeswift/makeswift/examples/basic-typescript
```
