# Lab - Set Up Catalyst and Makeswift

## Install the Catalyst Project

### Run the Installer

```shell
corepack enable pnpm && pnpm create @bigcommerce/catalyst@0.20.0 --gh-ref @bigcommerce/catalyst-core@makeswift-1.0.0-rc.3
```

### Run the Dev Server

```shell
cd /path/to/catalyst/project
pnpm run dev
```
