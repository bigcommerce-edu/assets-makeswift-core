# Lab - Set Up Catalyst and Makeswift

## Install the Catalyst Project

### Run the Installer

```shell
corepack enable pnpm && pnpm create @bigcommerce/catalyst@1.0.0 --gh-ref @bigcommerce/catalyst-makeswift@1.0.0
```

### Run the Dev Server

```shell
cd /path/to/catalyst/project
pnpm run dev
```
